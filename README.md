## Description
Jeu d'échecs en Python avec interface graphique pygame.

## Installation
```bash
git clone https://github.com/pseudo/chess-isart-2026.git
cd Satara-
pip install pygame
```

## Lancement
```bash
python main.py
```

## Contrôles
- Clic gauche : sélectionner et déplacer une pièce
- S : sauvegarder la partie
- L : charger une partie
